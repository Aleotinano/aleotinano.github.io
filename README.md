# Portafolio 2025

Portafolio personal desarrollado para presentar mis proyectos, experiencia y enfoque como desarrollador frontend.  
El objetivo principal del proyecto es ofrecer una experiencia clara, fluida y visualmente cuidada, priorizando la usabilidad, el rendimiento y una arquitectura moderna.

## Características

- Sitio construido con Astro para optimizar rendimiento y carga inicial
- Componentes interactivos desarrollados con React
- Animaciones y transiciones suaves con Framer Motion
- Estilos consistentes y escalables con Tailwind CSS
- Diseño completamente responsive
- Arquitectura modular orientada a mantenibilidad

## Stack tecnológico

- JavaScript
- Astro
- React
- Tailwind CSS
- Framer Motion
- PNPM para la gestión de dependencias
- Git para control de versiones

## Notas

Este portafolio representa mi enfoque actual en el desarrollo frontend, combinando diseño, arquitectura de componentes y experiencia de usuario.  
El proyecto se encuentra en evolución constante, incorporando mejoras a partir de feedback y nuevas ideas.
